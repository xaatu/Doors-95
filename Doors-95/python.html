I really enjoy working in Python. Up until now, I have mainly built calculators and some fairly basic music creation projects

Calculator

Below is a Calculator that performs some basic, and then some more advanced mathematical functions. This project helped me learn more about functions within Python, how to properly define them, and then how to create a display which could take user input.

Physics Calculator

Drawing on what I learned from the first Calculator, I decided to design a Physics Calculator, scripted in Python and running in Terminal. It started off with only a few functions, but with the help of Physics.info, I had a go at completing some more advanced formulae.

MIDI Generator

Below is the start of a MIDI Generator. The idea is that by pre-programming notes to keys, I should be able to generate a selection of those notes and export it as a .mid file which can then be imported into music production software. Eventually, it would be a stretch goal to create Generators specific to certain instruments, so that (in theory) I could write a song at the click of a button... or a few.
