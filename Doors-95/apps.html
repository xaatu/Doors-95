This is my first attempt at using the Flutter framework to create an IOS app. Originally this came from a 6 hour task set by a potential employer, but after completion of the 6 hours, I decided to flesh it out more.

"SHAPE Movies" Home Screen

GitHub

This is the home screen. It features a logo I designed, with a bold App Bar title and rounded button.

I designed the logo in GIMP.

The button directs the user to the "Movie Search" screen

The app features navigation to and from each page, in the form of buttons.

While the app will run on IOS and Mac, I have not yet fully tested it for android so I can't confirm.

"SHAPE Movies" Search Screen

GitHub

This is the Movie Search screen. It pulls data from the OMDB API and displays it in a column down the page.

The monochromatic design is consistent throughout the app, by using a theme.dart file to define theme data for the other files.

I also added error handling, in the form of movie quotes. When a user tries to search for a term that brings up an error, they instead get "Houston, we have a problem" - Apollo 13 (1995), followed by the error message "Try searching for a different title!"

"SHAPE Movies" Details Screen

GitHub

This is the Movie Details screen. It appears when the users clicks on a film, and displays the poster and film details in a full page.

The idea of this page was to eventually have a short plot of the film, and perhaps another data point pulled from the OMDB API.

If there is no associated poster for the film, the default flutter "Film" icon will show, and similarly with other film details "N/A" will appear.
